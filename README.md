# Easter-Egg-Game

프론트엔드 개발자가 숨겨놓은 이스터에그를 찾는 웹 게임입니다.

## Technologies

- Next.js 13
- NextUI v2
- Tailwind CSS
- Tailwind Variants
- TypeScript
- Framer Motion
- next-themes

## How to Use
```
git clone <url>

npm install

npm run dev
```
